Detailed Description

Basic socket, whose behavior is as follows. (a) The sender sends out a data packet, (b) the receiver acknowledges the packet by sending an ACK packet back, (c) the sender receives the ACK packet and responds with a new data packet if available.

This basic socket assumes a perfect medium.
