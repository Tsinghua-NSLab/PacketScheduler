◆ draw()

double ch.ethz.systems.netbench.ext.poissontraffic.ParetoDistribution.draw ( )

Inverse of the CDF of a Pareto distribution: CDF = 1 - (scale / x)^shape 1 - CDF = (scale / x)^shape (1 - CDF)^(1/shape) = scale / x x = scale / ((1 - CDF)^(1/shape))

Because outcome is drawn uniformly from [0, 1.0), (1 - CDF) is replaced with the outcome.

Returns
Value drawn from parametrized Pareto distribution
