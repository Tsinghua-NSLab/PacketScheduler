Detailed Description

Abstraction for a network device.

All nodes in the network are instances of this abstraction. It takes care of the definition of network connections and forces its subclasses to be able to handle packets it receives. A network device is a server iff it has a transport layer.

It enables additional modification of packets by placement of a intermediary in between the network device and the transport layer.
