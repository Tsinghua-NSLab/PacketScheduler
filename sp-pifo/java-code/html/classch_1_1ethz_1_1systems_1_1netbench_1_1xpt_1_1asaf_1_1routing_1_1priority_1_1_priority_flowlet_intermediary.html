Detailed Description

This is used to add priority to packets, when a flowlet gap is encountered not only do we change paths (as the UniformIntermediary) but we also modify this flow as lower priority.

This translates to: 1) Set flow id as low priority 2) Measure the amount of current window size 3) If we've sent the current window size without interuptions, bring back the regular priority 4) if not we reset the count
